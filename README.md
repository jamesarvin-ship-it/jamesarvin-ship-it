## 🏗️ What we're building

Mavricon is the **OS for construction** — five product lines on one connected platform:

| Product Line | Tools |
|---|---|
| 🏗️ **Operations** | Mail · Files · Forms · Issues · Schedule · Site Diary |
| ✅ **Quality** | Defects · Inspections · Procedures |
| 🦺 **Safety** | Hazards · Permits · SWMS · Toolbox · First Aid |
| 💰 **Commercials** | Budgets · Contracts · Variations · Costs · Movements |
| 🔧 **Aftercare** | Assets · Handover · Service |

Priced on annual construction volume, not seat count. AI woven through every tool — not bolted on.
